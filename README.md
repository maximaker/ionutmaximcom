## üì¶ Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/maximaker/ionutmaximcom.git
cd ionutmaximcom
```

### 2. Install dependencies
```bash
npm install
# or
pnpm install
```

### 3. Run the development server
```bash
npm run dev
```
Visit [http://localhost:3000](http://localhost:3000) to view the site.
